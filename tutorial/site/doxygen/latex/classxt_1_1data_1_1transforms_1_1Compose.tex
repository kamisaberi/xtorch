\doxysubsection{Detailed Description}
A class to compose multiple tensor transformations into a single callable pipeline. 

The \doxylink{classxt_1_1data_1_1transforms_1_1Compose}{Compose} class allows chaining of multiple transformation functions, each operating on a {\ttfamily torch\+::\+Tensor}, into a single operation. It is designed to facilitate preprocessing or augmentation of tensor data (e.\+g., images) by applying a sequence of transforms in the order they are provided. The transformations are stored as a vector of function objects and applied via the call operator.
