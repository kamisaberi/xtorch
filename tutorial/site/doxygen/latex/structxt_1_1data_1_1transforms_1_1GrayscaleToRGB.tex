\doxysubsection{Detailed Description}
A functor to convert a grayscale tensor to an RGB tensor. 

This struct provides a callable object that transforms a grayscale tensor, typically with a single channel (e.\+g., \mbox{[}H, W\mbox{]} or \mbox{[}1, H, W\mbox{]}), into an RGB tensor with three channels (e.\+g., \mbox{[}3, H, W\mbox{]}). The conversion is performed by replicating the grayscale channel across the RGB dimensions, suitable for preprocessing grayscale images in machine learning workflows using Lib\+Torch.
